# 🔢 Keith Number Detector

A Python program that detects **Keith numbers** using a sequence generated from the digits of the number.

---

## 📌 Description

A **Keith number** is an integer that appears in a sequence generated from its own digits.

The algorithm:

1. Calculate the number of digits `k`.
2. Use the digits of the number as the first `k` terms.
3. Generate the next terms by adding the last `k` terms.
4. Continue until the generated term is greater than or equal to the number.
5. If the generated term is equal to the original number, it is a Keith number.

---

# 🚀 Features

✨ The program can:

- 📥 Read a list of integers from the user
- 🔍 Check if numbers are Keith numbers
- 📦 Store detected Keith numbers in a second array
- 📊 Display Keith numbers grouped by their number of digits
- ✅ Validate user input
- 🧩 Use modular functions for better organization
